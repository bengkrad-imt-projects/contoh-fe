Dokumentasi Contoh Proyek Frontend

Dokumentasi ini merupakan referensi untuk memahami struktur dan cara kerja proyek contoh yang menggunakan ReactJS sebagai frontend. Proyek ini dapat dijadikan acuan dalam pengembangan proyek kalian dengan teknologi yang sama. Silakan gunakan dokumentasi ini untuk memahami bagaimana frontend berkomunikasi dengan backend serta bagaimana data dikelola di dalamnya. Dokumentasi ini berisi:

  • Cara menjalankan proyek
  • Bagaimana aplikasi bekerja secara garis besar
  • Bagian yang perlu diperhatikan dalam kode
  • Komunikasi dengan tim backend

1. Cara Menjalankan Proyek

Ikuti langkah-langkah berikut untuk menjalankan proyek frontend ini:

            git clone https://github.com/bengkrad-imt-projects/contoh-fe
            cd contoh-fe
            npm install
            npm start

2. Bagaimana Aplikasi Bekerja

Secara garis besar, alur kerja aplikasi adalah sebagai berikut:

  • Ketika halaman dimuat, aplikasi akan mengambil data jadwal dari backend melalui API.
  • Data yang diterima kemudian diproses dan disimpan dalam state menggunakan React hooks.
  • Pengguna dapat memilih jadwal kosong dengan mengklik checkbox yang tersedia.
  • Jika pengguna menyimpan perubahan, data akan dikirim kembali ke backend.

3. Bagian yang Perlu Diperhatikan

  • src/components/: Berisi komponen utama, termasuk Navbar.js dan PilihSchedule.js.
  • src/hooks/: Berisi useSchedule.js untuk mengelola state dan komunikasi dengan API.
  • src/utils/: Berisi Schedule.js yang mendefinisikan struktur data jadwal.

4. Komunikasi dengan Tim Backend

Aplikasi ini membutuhkan komunikasi dengan backend dalam beberapa aspek:

  • Mengambil data jadwal saat aplikasi pertama kali dimuat.
  • Mengirim data yang telah diubah oleh pengguna kembali ke backend.

Format Data yang Dikirim dan Diterima

Contoh format request yang dikirim ke backend saat menyimpan data:

            {
                "id": "0",
                "jadwal": [
                    { "Senin": { "07:00 - 09.00": 1, "08:00 - 10.00": 0 } },
                    { "Selasa": { "07:00 - 09.00": 0, "08:00 - 10.00": 1 } }
                ]
            }

Respons yang diterima dari backend saat mengambil data:

            {
                "selected-schedule": [
                    {
                        "id": "0",
                        "jadwal": [
                            { "Senin": { "07:00 - 09.00": 1, "08:00 - 10.00": 0 } },
                            { "Selasa": { "07:00 - 09.00": 0, "08:00 - 10.00": 1 } }
                        ]
                    }
                ]
            }

Bagian Kode yang Mengirim dan Menerima Data

Di dalam useSchedule.js, data diambil dari backend menggunakan:

            const response = await axios.get(apiURL);

Data dikirim kembali ke backend menggunakan:

            await axios.post(apiURL, payload, { headers: { "Content-Type": "application/json" } });

Bagian ini harus disesuaikan dengan backend yang digunakan. Tim frontend perlu berdiskusi dengan tim backend mengenai:

  • Endpoint API yang digunakan
  • Format data yang dikirim dan diterima
  • Metode HTTP yang digunakan

Semoga dokumentasi ini membantu dalam pengembangan proyek kalian. Jika ada pertanyaan atau hal yang perlu didiskusikan lebih lanjut, jangan ragu untuk berkomunikasi dengan tim terkait.
